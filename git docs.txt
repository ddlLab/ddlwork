
fetch - подгрузить последние обновления
pull - забирает последние данные по ветки
commit - фиксируем измения на компе
push - отправка фиксированых изменений 
merge - слияние двух веток в одну
checkout - переключение на другую ветку
reset - откатывает удаляет изменения

tag - подсказка

branch - ветка 
create branch - создать ветку(локально)

gitignore - файл с правилами игнорирования по проекту

cherry pick - более тонкий вид мержа. перетаскивание конкретного
